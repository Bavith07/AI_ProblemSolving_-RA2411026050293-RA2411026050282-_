ℹ How It Works

BFS (Breadth-First Search)

Explores all neighbors at the current depth before moving deeper. Uses a queue (FIFO). Always finds the shortest path in unweighted graphs.

DFS (Depth-First Search)

Explores as deep as possible along each branch before backtracking. Uses a stack (LIFO). May find a longer path than optimal.
